# Threshold 1: Foundation

*Presence, perception, honesty, and the agreements that make practice safe.*

This is where it begins. Before depth, before navigating challenge, before any of the more ambitious work two people might do together, there is the simpler question of whether we can actually be here, with each other, honestly. Can we land in the same room? Can we sense each other accurately? Can we say true things and trust they will be received? Can we set down what we cannot give and ask for what we need? These are not given. They are practiced.

This threshold gathers the foundational capacities — presence, consent, accountability, listening, attunement, authenticity, vulnerability, trust, safety — and the agreements that make all of them possible. None of the later thresholds work without these. Every return to the arc returns here first, and the practice is different the second time, and the third.

Four sub-thresholds, twenty practices.
